Portfolio Layout Testing Guide

1. Check Tech Stack Section

Visit http://localhost:3001 and scroll to the Technology Stack section.

  • ✓ Title should be centered
  • ✓ Grid should have 5 columns on desktop, 3 on tablet, 2 on mobile
  • ✓ Cards should have consistent height (120px)
  • ✓ Icons should be 32px
  • ✓ Proper padding on left and right

2. Check Project Cards

Look at the Featured Projects section.

  • ✓ All cards should maintain 16:9 aspect ratio
  • ✓ Images should fill the container properly
  • ✓ Consistent padding around content
  • ✓ TiltFx effect should work on hover

3. Check Overall Layout

  • ✓ Content should have equal padding on left and right
  • ✓ Maximum width of 1440px centered on large screens
  • ✓ No horizontal scrolling
  • ✓ Proper spacing between sections

4. Responsive Testing

Use browser dev tools to test at these breakpoints:

  • Mobile: 375px, 414px
  • Tablet: 768px, 1024px
  • Desktop: 1280px, 1440px, 1920px

5. Developer Tools Inspection

Right-click on elements and inspect:

  • Check computed styles for padding values
  • Verify grid layout is working correctly
  • Look for any CSS conflicts in the styles panel
  • Check if Once UI data attributes are applied
